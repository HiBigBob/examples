# Todo Front-End - Vue.Js

## Prerequisites

You will need the following things properly installed on your computer.

* [Node.js](http://nodejs.org/) (with NPM)

## Installation

* `cd front-vuejs`
* `npm install`

## Running / Development

* `npm run dev`
* Visit your app at [http://localhost:8080](http://localhost:8080).

### Building production

* `npm run build`
