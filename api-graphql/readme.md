# Todo Api - Node.Js

## Prerequisites

You will need the following things properly installed on your computer.

* [Node.js](http://nodejs.org/) (with NPM)

## Installation

* `cd api-node`
* `npm install`

## Running / Development

* `npm start`
* Visit your app at [http://localhost:3000](http://localhost:3000).

### Running Tests

* `npm run test`
